## 整体设计风格
- **深色主题**（适用于工业平板/盒子，与系统设置风格统一）
- **主色调**：科技蓝 `#2196F3`，辅以白色文字
- **字体**：Droid Sans / Roboto，标题加粗
- **布局**：单 Activity 多 Fragment 或单界面多状态切换

## 1. 主界面（空闲状态 / 检查更新入口）

```
┌──────────────────────────────────────┐
│  [返回]  固件升级              [⋮]   │  ← Toolbar
├──────────────────────────────────────┤
│                                      │
│    当前服务器地址                      │
│  ┌────────────────────────────────┐  │
│  │ http://192.168.1.100:8080  [✎] │  │  ← 可点击编辑，右侧按钮跳转修改
│  └────────────────────────────────┘  │
│                                      │
│    当前版本                           │
│    [     V1.0.0 (eng.20260501)    ]  │  ← 只读展示框
│                                      │
│                                      │
│         ┌──────────────────┐        │
│         │    检查更新      │        │  ← 主要操作按钮（蓝色填充）
│         └──────────────────┘        │
│                                      │
│    [ 扫码配置 ]     [ 手动输入 ]     │  ← 底部两个次要按钮
│                                      │
└──────────────────────────────────────┘
```
**说明**：
- 服务器地址一行可点击，点击后弹出编辑对话框或跳转设置页。
- 当前版本从系统属性读取。
- 若从未配置服务器，隐藏地址框，显示“请先设置服务器地址”并突出扫码/输入按钮。

## 10. 交互流程串联图

```
[启动APP] → (检查服务器地址) → 无地址 → [扫码/手动输入] → 保存成功 → 主界面
                ↓ 有地址
              [主界面] → 点击“检查更新”
                ↓
         (请求update_info.txt)
                ↓
    ┌──────────┴──────────┐
    │ 无更新               │ 有更新
    │ [Toast:已是最新]     │ [弹窗：发现新版本]
    └─────────────────────┘         ↓
                            点击“立即更新”
                                    ↓
                              [下载界面] (可暂停/续传, 后台Service)
                                    ↓
                              下载完成 → [准备中...]
                                    ↓
                              [系统升级中...] (进度回调)
                                    ↓
                              升级成功 → [重启弹窗]
                                    ↓
                        ┌───────┴───────┐
                        │ 立即重启        │ 取消
                        │ (执行reboot)    │ → [主界面-重启待命]
                        └───────────────┘
```
